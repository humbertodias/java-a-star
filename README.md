# Java A*

A* using Java


## Prerequires

1. Git 2.6+
2. Maven 3+
3. Java 8+


## How to Play

Clone

```
git clone https://github.com/humbertodias/java-a-star.git
```

Inside the folder

```
cd java-a-star
```

Run

```
mvn compile exec:java -Dexec.mainClass="astar.Main"
```
